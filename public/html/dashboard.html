ü§ñ Colby GitHub Bot - Command Reference

Colby is an AI-powered GitHub bot that helps with code reviews, issue management, and workflow automation. Use these commands in GitHub PR comments.

üöÄ Installation & Setup

/colby help
Shows this help message with all available commands.

üîß Implementation Commands

/colby implement
Applies code suggestions from review comments. Analyzes suggestion blocks and creates commits with the proposed changes.
/apply
Legacy command - same as /colby implement

üìù Issue Management

/colby create issue
Creates a GitHub issue with AI-generated title and comprehensive description. Analyzes conversation context, code suggestions, and file references to create professional, actionable issues that preserve the full discussion context.
/colby create issue and assign to copilot
Creates a new issue with enhanced AI analysis and assigns it to @copilot for automated handling. Includes smart labeling and context preservation.

üìö Knowledge Management

/colby bookmark this suggestion
Saves code suggestions as best practices in the knowledge base. AI automatically categorizes by technology stack and context.
/colby extract suggestions
Extracts all suggestions from Gemini/AI code reviews and prepares them for automated implementation via Codex.

üìä Analysis & Summaries

/summarize
Generates an AI-powered summary of the pull request, including changed files and impact analysis.
